1. ให้ทำการ pull git url ลงเครื่อง ณ ตำแหน่ง c:/รหัสนักศึกษาแบบเต็ม/

2. แก้ไขรูป และชื่อ ให้เป็นข้อมูลตนเอง (ใช้รูปจาก rms) 

3. push ข้อมูลชุดใหม่ เจ้าสู่ remote repository ของนักเรียน โดยใช้ชื่อเป็น รหัส นร. 3 ตัวสุดท้าย

4. โพส git url ของตนเอง ลงโพส ต่อจาก โพสข้อสอบ ในคลาสรูม

5. อย่าลบ หรือปิดหน้า cmd นะ จะตรวจขั้นตอนด้วย
